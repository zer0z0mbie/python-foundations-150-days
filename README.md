# Python Foundations – 150 Days

This repository documents my structured journey to strengthen my Python programming fundamentals over 150 days.

While I have prior exposure to Python through courses, tutorials, and university studies, this project focuses on reinforcing core concepts from the ground up, building consistency, and developing a clean, professional coding practice.

---

## 🎯 Objectives

- Reinforce fundamental programming concepts through daily practice
- Build discipline and consistency by coding every day
- Progress from simple constructs to more complex problem-solving
- Maintain a clean and structured GitHub portfolio
- Improve code clarity, readability, and organisation

---

## 🧠 Approach

- One small program per day
- Gradual increase in complexity
- Focus on understanding over speed
- Regular review and refactoring
- Each file includes a short learning reflection
